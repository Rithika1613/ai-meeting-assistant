# ai-meeting-assistantAI Meeting Assistant
About the Project

This AI Meeting Assistant is a web-based tool that helps generate meeting summaries, key points, and action items using Generative AI.

Features
Convert meeting input into structured summary
Extract key points from meetings
Identify action items automatically
AI-powered response generation
Simple and user-friendly interface
Technologies Used
Python
Flask
Generative AI API (Gemini/OpenAI)
HTML, CSS
